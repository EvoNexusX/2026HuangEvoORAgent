**数学模型**

**集合：**  
- 仓库集合 \(I = \{1,2,3,4,5,6\}\)，分别对应 Verona, Perugia, Rome, Pescara, Taranto, Lamezia。  
- 港口集合 \(J = \{1,2,3,4,5\}\)，分别对应 Genoa, Venice, Ancona, Naples, Bari。

**参数：**  
- \(s_i\)：仓库 \(i\) 的空集装箱供应量，具体值：  
  \(s_1 = 10, s_2 = 12, s_3 = 20, s_4 = 24, s_5 = 18, s_6 = 40\)。  
- \(d_j\)：港口 \(j\) 的空集装箱需求量，具体值：  
  \(d_1 = 20, d_2 = 15, d_3 = 25, d_4 = 33, d_5 = 21\)。  
- \(\text{dist}_{ij}\)：从仓库 \(i\) 到港口 \(j\) 的距离（公里），由给定表格定义。  
- \(c_{ij}\)：从仓库 \(i\) 到港口 \(j\) 运输一个集装箱的成本，计算公式为 \(c_{ij} = 30 \cdot \text{dist}_{ij}\)。

**决策变量：**  
- \(x_{ij} \geq 0\)：从仓库 \(i\) 运往港口 \(j\) 的集装箱数量（可视为连续变量，最优解自动取整）。

**目标函数：**  
最小化总运输成本：  
\[
\min \sum_{i \in I} \sum_{j \in J} c_{ij} x_{ij}
\]

**约束条件：**  
1. 供应约束（每个仓库运出的集装箱不超过其库存）：  
   \[
   \sum_{j \in J} x_{ij} \leq s_i, \quad \forall i \in I
   \]  
2. 需求约束（每个港口收到的集装箱满足其需求）：  
   \[
   \sum_{i \in I} x_{ij} = d_j, \quad \forall j \in J
   \]  
3. 非负约束：  
   \[
   x_{ij} \geq 0, \quad \forall i \in I, j \in J
   \]

根据给定的数学模型，设计以下Gurobi求解算法方案：

- **港口集合** `J = [1,2,3,4,5]`，对应需求量列表 `d = [20,15,25,33,21]`。

  ```
  ]
  ```
- **成本矩阵** `c`：计算每个 `c[i][j] = 30 * dist[i][j]`。

### 3. 创建决策变量
- 定义变量 `x[i,j]`，对每个 `i in I`、`j in J`：
  ```
  x = model.addVars(I, J, lb=0, vtype=GRB.CONTINUOUS, name="x")
  ```
  （注：`GRB.CONTINUOUS`可简写为`'C'`；由于运输问题的全幺模性，连续变量可保证整数最优解。）

### 4. 设置目标函数
- 最小化总运输成本：

  ```
  model.setObjective(gp.quicksum(c[i-1][j-1] * x[i,j] for i in I for j in J), GRB.MINIMIZE)
  ```

- **需求约束**（每个港口接收量严格等于需求）：
  ```
  for j in J:
      model.addConstr(gp.quicksum(x[i,j] for i in I) == d[j-1], name=f"Demand_{j}")
  ```

 ```python
warehouses = ["Verona", "Perugia", "Rome", "Pescara", "Taranto", "Lamezia"]
ports = ["Genoa", "Venice", "Ancona", "Naples", "Bari"]

supply = [10, 12, 20, 24, 18, 40]
demand = [20, 15, 25, 33, 21]

# 距离矩阵（公里）
distances = [
    [290, 115, 355, 715, 810],
    [380, 340, 165, 380, 610],

costs = [[dist * cost_per_km for dist in row] for row in distances]

# 创建变量

# 设置目标函数

    gp.quicksum(
        costs[i][j] * x[i, j]
        for i in range(len(warehouses))
        for j in range(len(ports))
    ),
    GRB.MINIMIZE
)

    model.addConstr(
        gp.quicksum(x[i, j] for j in range(len(ports))) <= supply[i],
        name=f"Supply_{warehouses[i]}"
    )

# 添加需求约束
for j in range(len(ports)):
    model.addConstr(
        gp.quicksum(x[i, j] for i in range(len(warehouses))) == demand[j],
        name=f"Demand_{ports[j]}"
    )

# 求解模型
model.optimize()

# 输出结果
if model.status == GRB.OPTIMAL:
    print(f"最优总成本: {model.objVal:.2f} 欧元")
    print("\n最优运输方案:")
    for i in range(len(warehouses)):
        for j in range(len(ports)):

                print(f"{warehouses[i]} -> {ports[j]}: {x[i, j].x:.0f} 个集装箱")
    
    # 计算并输出卡车数量
    total_containers = sum(demand)
    trucks_needed = (total_containers + 1) // 2  # 向上取整
    print(f"\n需要卡车数量: {trucks_needed} 辆（每辆最多载2个集装箱）")
else:
    print("未找到最优解")
```

 ```python
model.setObjective(5*T + 10*A + 8*B + 7*R, GRB.MAXIMIZE)

# 添加约束
# 木材约束

model.addConstr(6*T + 3*A + 5*B + 4*R <= 500, "Steel")

model.addConstr(y_T + y_R <= 1, "Truck_Train_Exclusive")
# 逻辑约束：生产船则生产飞机
model.addConstr(y_B <= y_A, "Boat_Implies_Airplane")
# 数量约束：船的数量不超过火车数量
model.addConstr(B <= R, "Boat_leq_Train")

model.addConstr(A <= 44 * y_A, "Ind_A")
model.addConstr(B <= 59 * y_B, "Ind_B")
model.addConstr(R <= 89 * y_R, "Ind_R")

model.optimize()

# 输出结果

    print(f"最优目标值: ${model.ObjVal:.2f}")
    print("生产数量:")
    print(f"  卡车: {int(T.X)}")

    print("是否生产:")
    print(f"  卡车: {int(y_T.X)}")
    print(f"  飞机: {int(y_A.X)}")
    print(f"  船: {int(y_B.X)}")
    print(f"  火车: {int(y_R.X)}")
else:
    print("未找到最优解")
```
