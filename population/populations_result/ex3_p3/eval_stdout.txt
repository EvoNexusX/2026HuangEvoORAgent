=============== num 7 ==================
An Italian transportation company needs to move some empty containers from its 6 warehouses (located in Verona, Perugia, Rome, Pescara, Taranto, and Lamezia) to major national ports (Genoa, Venice, Ancona, Naples, Bari). The container inventory at the warehouses is as follows:

|  | Empty Containers |
|:---:|:---:|
| Verona | 10 |
| Perugia | 12 |
| Rome | 20 |
| Pescara | 24 |
| Taranto | 18 |
| Lamezia | 40 |

The demand at the ports is as follows:

|  | Container Demand |
|:---:|:---:|
| Genoa | 20 |
| Venice | 15 |
| Ancona | 25 |
| Naples | 33 |
| Bari | 21 |

The transport is carried out by a fleet of trucks. The cost to transport each container is proportional to the distance traveled by the trucks, with a rate of 30 euros per kilometer. Each truck can carry up to 2 containers. The distances are as follows:

|  | Genoa | Venice | Ancona | Naples | Bari |
|:---:|:---:|:---:|:---:|:---:|:---:|
| Verona | $290 \mathrm{~km}$ | $115 \mathrm{~km}$ | $355 \mathrm{~km}$ | $715 \mathrm{~km}$ | $810 \mathrm{~km}$ |
| Perugia | $380 \mathrm{~km}$ | $340 \mathrm{~km}$ | $165 \mathrm{~km}$ | $380 \mathrm{~km}$ | $610 \mathrm{~km}$ |
| Rome | $505 \mathrm{~km}$ | $530 \mathrm{~km}$ | $285 \mathrm{~km}$ | $220 \mathrm{~km}$ | $450 \mathrm{~km}$ |
| Pescara | $655 \mathrm{~km}$ | $450 \mathrm{~km}$ | $155 \mathrm{~km}$ | $240 \mathrm{~km}$ | $315 \mathrm{~km}$ |
| Taranto | $1010 \mathrm{~km}$ | $840 \mathrm{~km}$ | $550 \mathrm{~km}$ | $305 \mathrm{~km}$ | $95 \mathrm{~km}$ |
| Lamezia | $1072 \mathrm{~km}$ | $1097 \mathrm{~km}$ | $747 \mathrm{~km}$ | $372 \mathrm{~km}$ | $333 \mathrm{~km}$ |

Write a mathematical program to find the minimum cost transportation policy and solve it using COPTPY.
-------------
Using Agent mode (three-stage: modeling + design + code generation)
【Stage 1 - Problem Analysis & Modeling】:
 **Mathematical Model**

**Sets:**
- \(I\): set of warehouses (6 warehouses: Verona, Perugia, Rome, Pescara, Taranto, Lamezia)
- \(J\): set of ports (5 ports: Genoa, Venice, Ancona, Naples, Bari)

**Parameters:**
- \(s_i\): supply of empty containers at warehouse \(i\), \(i \in I\)
- \(d_j\): demand for containers at port \(j\), \(j \in J\)
- \(\text{dist}_{ij}\): distance (in km) from warehouse \(i\) to port \(j\), \(i \in I\), \(j \in J\)
- \(c\): cost per km per truck = 30 euros/km
- \(\text{CAP}\): truck capacity = 2 containers

**Decision Variables:**
- \(x_{ij}\): integer, number of containers transported from warehouse \(i\) to port \(j\), \(i \in I\), \(j \in J\)
- \(y_{ij}\): integer, number of trucks used from warehouse \(i\) to port \(j\), \(i \in I\), \(j \in J\)

**Objective Function:**
Minimize total transportation cost:
\[
\min \sum_{i \in I} \sum_{j \in J} c \cdot \text{dist}_{ij} \cdot y_{ij}
\]

**Constraints:**
1. **Supply constraints (do not exceed available containers):**
   \[
   \sum_{j \in J} x_{ij} \le s_i, \quad \forall i \in I
   \]
2. **Demand constraints (exactly meet port demand):**
   \[
   \sum_{i \in I} x_{ij} = d_j, \quad \forall j \in J
   \]
3. **Truck capacity constraints:**
   \[
   x_{ij} \le \text{CAP} \cdot y_{ij}, \quad \forall i \in I, j \in J
   \]
4. **Non-negativity and integrality:**
   \[
   x_{ij} \ge 0, \quad \text{integer}, \quad \forall i \in I, j \in J
   \]
   \[
   y_{ij} \ge 0, \quad \text{integer}, \quad \forall i \in I, j \in J
   \]

---

**Solution using COPTPY (Python with COPT)**

```python
import coptpy as cp

# Data
warehouses = ['Verona', 'Perugia', 'Rome', 'Pescara', 'Taranto', 'Lamezia']
ports = ['Genoa', 'Venice', 'Ancona', 'Naples', 'Bari']

supply = {'Verona': 10, 'Perugia': 12, 'Rome': 20, 'Pescara': 24, 'Taranto': 18, 'Lamezia': 40}
demand = {'Genoa': 20, 'Venice': 15, 'Ancona': 25, 'Naples': 33, 'Bari': 21}

# Distance matrix (km)
dist = {
    ('Verona', 'Genoa'): 290,
    ('Verona', 'Venice'): 115,
    ('Verona', 'Ancona'): 355,
    ('Verona', 'Naples'): 715,
    ('Verona', 'Bari'): 810,
    ('Perugia', 'Genoa'): 380,
    ('Perugia', 'Venice'): 340,
    ('Perugia', 'Ancona'): 165,
    ('Perugia', 'Naples'): 380,
    ('Perugia', 'Bari'): 610,
    ('Rome', 'Genoa'): 505,
    ('Rome', 'Venice'): 530,
    ('Rome', 'Ancona'): 285,
    ('Rome', 'Naples'): 220,
    ('Rome', 'Bari'): 450,
    ('Pescara', 'Genoa'): 655,
    ('Pescara', 'Venice'): 450,
    ('Pescara', 'Ancona'): 155,
    ('Pescara', 'Naples'): 240,
    ('Pescara', 'Bari'): 315,
    ('Taranto', 'Genoa'): 1010,
    ('Taranto', 'Venice'): 840,
    ('Taranto', 'Ancona'): 550,
    ('Taranto', 'Naples'): 305,
    ('Taranto', 'Bari'): 95,
    ('Lamezia', 'Genoa'): 1072,
    ('Lamezia', 'Venice'): 1097,
    ('Lamezia', 'Ancona'): 747,
    ('Lamezia', 'Naples'): 372,
    ('Lamezia', 'Bari'): 333,
}

c = 30  # cost per km per truck
CAP = 2  # truck capacity (containers)

# Create model
env = cp.Envr()
model = env.createModel("ContainerTransportation")

# Decision variables
x = {}  # containers
y = {}  # trucks
for i in warehouses:
    for j in ports:
        x[i,j] = model.addVar(vtype=cp.COPT.INTEGER, lb=0, name=f"x_{i}_{j}")
        y[i,j] = model.addVar(vtype=cp.COPT.INTEGER, lb=0, name=f"y_{i}_{j}")

# Supply constraints
for i in warehouses:
    model.addConstr(cp.quicksum(x[i,j] for j in ports) <= supply[i], name=f"Supply_{i}")

# Demand constraints
for j in ports:
    model.addConstr(cp.quicksum(x[i,j] for i in warehouses) == demand[j], name=f"Demand_{j}")

# Truck capacity constraints
for i in warehouses:
    for j in ports:
        model.addConstr(x[i,j] <= CAP * y[i,j], name=f"Capacity_{i}_{j}")

# Objective
model.setObjective(cp.quicksum(c * dist[i,j] * y[i,j] for i in warehouses for j in ports), cp.COPT.MINIMIZE)

# Solve
model.solve()

# Output results
if model.status == cp.COPT.OPTIMAL:
    print("Optimal solution found.")
    print(f"Total cost: {model.objval:.2f} euros")
    print("\nContainer flow (x_ij):")
    for i in warehouses:
        for j in ports:
            val = x[i,j].x
            if val > 0:
                print(f"  {i} -> {j}: {val} containers")
    print("\nTruck usage (y_ij):")
    for i in warehouses:
        for j in ports:
            val = y[i,j].x
            if val > 0:
                print(f"  {i} -> {j}: {val} trucks (distance {dist[i,j]} km, cost {c * dist[i,j] * val:.2f} euros)")
else:
    print("No optimal solution found.")
```

**Note:** Ensure that the COPT (Cardinal Optimizer) Python library (`coptpy`) is installed. If not, install it via `pip install coptpy`. The above code defines and solves the mixed-integer linear programming model. The solution will provide the optimal container flows and truck assignments that minimize total transportation cost.
